# Как переключиться на Sepolia Testnet в MetaMask

## Пошаговая инструкция:

### 1. Откройте MetaMask
- Нажмите на расширение MetaMask в браузере

### 2. Переключитесь на сеть
- В верхней части MetaMask вы увидите название текущей сети (например, "Ethereum Mainnet")
- Нажмите на это название

### 3. Выберите Sepolia Testnet
- В выпадающем списке найдите "Sepolia Testnet"
- Если вы не видите Sepolia в списке, нажмите "Show test networks" (Показать тестовые сети)
- Затем выберите "Sepolia Testnet"

### 4. Если Sepolia Testnet отсутствует в списке
Если вы не видите Sepolia Testnet в списке доступных сетей:

1. Нажмите на название текущей сети
2. Нажмите "Add network" (Добавить сеть) внизу
3. Нажмите "Add a network manually" (Добавить сеть вручную)
4. Заполните следующие данные:

**Название сети:**
```
Sepolia Test Network
```

**RPC URL:**
```
https://sepolia.infura.io/v3/YOUR_INFURA_KEY
```
Или используйте публичные RPC:
```
https://rpc.sepolia.org
```

**Chain ID:**
```
11155111
```

**Символ валюты:**
```
ETH
```

**Block Explorer URL (опционально):**
```
https://sepolia.etherscan.io
```

5. Нажмите "Save" (Сохранить)

### 5. Проверка переключения
После переключения вы должны увидеть:
- В верхней части MetaMask: "Sepolia Test Network"
- Ваш баланс будет показываться в Sepolia ETH (тестовые ETH)
- Адрес вашего кошелька останется прежним

### 6. Получение тестовых ETH для Sepolia
Для получения бесплатных тестовых ETH на Sepolia:

1. **Sepolia Faucet от Alchemy:**
   - Перейдите на: https://sepoliafaucet.com/
   - Введите адрес вашего кошелька
   - Получите 0.5 Sepolia ETH бесплатно (каждые 24 часа)

2. **Infura Sepolia Faucet:**
   - Перейдите на: https://www.infura.io/faucet/sepolia
   - Войдите или зарегистрируйтесь
   - Получите тестовые ETH

3. **PoW Faucet:**
   - Перейдите на: https://sepolia-faucet.pk910.de/
   - Выполните задание (mining) для получения ETH

### 7. Получение тестовых USDT на Sepolia
Для получения тестовых USDT:

1. Найдите тестовый контракт USDT на Sepolia:
   - Адрес контракта: `0xaA8E23Fb1079EA71e0a56F48a2aA51851D8433D0` (это может измениться, проверьте актуальный)
   
2. Используйте faucet для ERC-20 токенов на Sepolia:
   - Некоторые faucets предоставляют тестовые токены, но их может быть сложно найти
   - Альтернатива: используйте тестовый DEX на Sepolia для обмена ETH на USDT

3. **Или протестируйте с ETH:**
   - Для тестирования функционала покупки вы можете использовать ETH вместо USDT
   - Просто выберите "ETH" в выборе криптовалюты на странице покупки

### Важные замечания:

⚠️ **Внимание:**
- Тестовые сети НЕ используют реальные деньги
- Все транзакции на тестовых сетях бесплатны (нужны только тестовые ETH для газа)
- Тестовые токены не имеют реальной стоимости
- Для production используйте Mainnet

✅ **Преимущества тестирования:**
- Можно экспериментировать без риска
- Быстрое подтверждение транзакций
- Бесплатные тестовые токены

### Возврат на Mainnet
Чтобы вернуться на Ethereum Mainnet:
1. Нажмите на название сети (Sepolia Test Network)
2. Выберите "Ethereum Mainnet" из списка
