# Как запустить Frontend

## Способ 1: Через командную строку (рекомендуется)

1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку проекта:
   ```
   cd C:\Users\admin\Desktop\dex_lottery\frontend
   ```
3. Запустите сервер:
   ```
   npm run dev
   ```

4. Откройте браузер и перейдите на: **http://localhost:3000**

## Способ 2: Через bat-файл

1. Дважды кликните на файл `start-frontend.bat` в корне проекта
2. Дождитесь сообщения "Ready - started server on 0.0.0.0:3000"
3. Откройте браузер и перейдите на: **http://localhost:3000**

## Если сервер не запускается

1. Убедитесь, что установлены зависимости:
   ```
   cd frontend
   npm install
   ```

2. Проверьте, что порт 3000 свободен:
   - Если порт занят, Next.js автоматически предложит использовать другой порт (например, 3001)
   - Смотрите сообщение в консоли

3. Проверьте ошибки в консоли - они покажут, что не так

## Примечание

Если контракты еще не развернуты, сайт все равно должен открыться, но некоторые функции (инвестирование) будут недоступны. Это нормально - интерфейс должен работать.
