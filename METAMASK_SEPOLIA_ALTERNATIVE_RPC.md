# Альтернативные RPC endpoints для Sepolia

## Если `https://rpc.sepolia.org` не работает, попробуйте:

### Вариант 1: PublicNode (рекомендуется)
```
https://ethereum-sepolia-rpc.publicnode.com
```

### Вариант 2: Ankr
```
https://rpc.ankr.com/eth_sepolia
```

### Вариант 3: Sepolia Gateway
```
https://sepolia.gateway.tenderly.co
```

### Вариант 4: 1RPC
```
https://1rpc.io/sepolia
```

### Вариант 5: LlamaNodes (бесплатный)
```
https://eth-sepolia.g.alchemy.com/v2/demo
```

### Вариант 6: Infura (публичный demo)
```
https://sepolia.infura.io/v3/9aa3d95b3bc440fa88ea12eaa4456161
```

## Как изменить RPC в MetaMask:

1. Откройте MetaMask
2. Нажмите на название сети "Sepolia" вверху
3. Нажмите на иконку настроек (⚙️) рядом с сетью
4. Или нажмите "Edit" (Редактировать) / "Изменить сеть"
5. В поле "RPC URL" введите один из альтернативных URLs выше
6. Нажмите "Save" (Сохранить)
